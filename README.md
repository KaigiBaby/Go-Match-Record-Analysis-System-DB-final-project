# Go Match Analysis System

A Visual Basic based database application designed to manage and analyze Go (Weiqi) match records.

## Features
- **Match Tracking**: Record details of every game played in the academy.
- **Data Analytics**: Analyze player performance and match statistics.
- **Database Driven**: Efficient storage and retrieval of historical game data.

## Requirements
- Visual Studio (with VB6 or VB.NET support depending on your version)
- Compatible Windows OS

## How to Run
1. Clone this repository.
2. Open the `.vbp` or `.sln` file in your Visual Studio environment.
3. Ensure the database connection string is correctly configured.
4. Build and Run the project.

# Go Match Record Analysis System
### 圍棋對弈紀錄管理系統 (資料庫期末專案)

## 專案簡介 (Project Overview)
本專案為資料庫課程期末作品，使用 **Visual Basic** 開發，旨在提供棋院一個系統化管理對弈紀錄、學員戰績與數據分析的工具。

## 主要功能 (Core Features)
* **對弈紀錄 (Match Recording)**: 記錄每場棋局的對手、日期、勝負結果。
* **數據分析 (Data Analytics)**: 自動統計學員的對局與相關資料。
* **資料庫管理 (Database)**: 結構化儲存所有棋譜與選手資訊。

## 開發工具 (Tech Stack)
* **Language**: Visual Basic (VB)
* **Database**: MySQL

## 如何執行 (Getting Started)
1. 下載本專案所有檔案。
2. 使用 Visual Studio 打開 `.vbp` (或 `.sln`) 專案檔。
3. 確保資料庫路徑設定正確後即可執行。
